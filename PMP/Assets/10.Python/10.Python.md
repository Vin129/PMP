# 关键字

## as

替代，适用于简写

```py
import xxxxx as x 
```



## with

使用with后不管with中的代码出现什么错误，都会进行对当前对象进行清理工作。

例如file的file.close()方法，无论with中出现任何错误，都会执行file.close（）方法
