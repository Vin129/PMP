## UnityEditor

### EditorWindow 生命周期

| OnEnable          | 打开界面                     |
| ----------------- | ---------------------------- |
| OnFocus           | 被聚焦时                     |
| OnInspectorUpdate | 属性界面更新时，几乎一直更新 |
| OnProjectChange   | 当项目发生更改时             |
| OnSelectionChange | 当选择发生更改时             |
| OnHierarchyChange | 场景层次发生改变时           |
| OnGUI             | 当渲染UI时                   |
| OnLostFocus       | 丢失聚焦时                   |
| OnDisable         | 隐藏时                       |
| OnDestroy         | 关闭时                       |

***
